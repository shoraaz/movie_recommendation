# Movie Recommendation

> 📦 **Archived learning project.** Kept public for reference; not actively maintained.

## What this is
A recommendation-system exercise on the MovieLens ratings data, worked through
across notebooks — building up from data exploration to similarity-based
recommendations.

## Contents
- `Notebook 1–3.ipynb` — exploration → recommender build
- `ratings.csv`, `movies_small.csv` — MovieLens-style data

## What it demonstrates
- User–item rating data handling at scale
- Collaborative-filtering / similarity recommendation fundamentals
